# Code Analysis Skill

## Steps

1. Run `analyze_structure.py` to summarize repository/project structure quality.
2. Run `analyze_model.py` to infer model architecture and technical characteristics.
3. Run `extract_benchmarks.py` to extract benchmark-related clues from metadata.
4. Return analysis blocks for report generation.
