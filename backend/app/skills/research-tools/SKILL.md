# Research Tools Skill

## Steps

1. Use `fetch_web_page.py` to retrieve web content, title, and useful links.
2. Use `list_local_files.py` to enumerate project files in a target directory.
3. Use `read_local_file.py` to read specific file ranges for evidence extraction.
4. Return structured results to the agent runtime.
